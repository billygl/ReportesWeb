 * ReportesWebMN-javaweb.zip
  * contiene los archivos de Eclipse como proyecto Java Web
  * descomprimir y usar.

 * ReportesWebMN-maven.zip
  * contiene los archivos de Eclipse como proyecto Maven
  * descomprimir y usar.

 * Las librerías están configuradas como librerías de usuario
  * Están ubicadas en un repositorio local de librerías.
  * Para MyBatis
   * mybatis-3.2.2.jar
   * mybatis-spring-1.2.0.jar
   * aopalliance.jar (usada por spring tx)
   * spring-jdbc-4.1.0.RELEASE.jar
   * spring-tx-4.1.0.RELEASE.jar
  * JSON - Java
   * jackson-annotations-2.2.0.jar
   * jackson-core-2.2.0.jar
   * jackson-databind-2.2.0.jar
  * Logging
   * log4j-1.2.17.jar
  * Spring
   * spring-aop-4.1.0.RELEASE.jar
   * spring-beans-4.1.0.RELEASE.jar
   * spring-context-4.1.0.RELEASE.jar
   * spring-core-4.1.0.RELEASE.jar
   * spring-expression-4.1.0.RELEASE.jar
   * spring-web-4.1.0.RELEASE.jar
   * spring-webmvc-4.1.0.RELEASE.jar
  * Spring web sockets
   * spring-messaging-4.1.0.RELEASE.jar
   * spring-websocket-4.1.0.RELEASE.jar
